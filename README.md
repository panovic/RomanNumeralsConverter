# Roman Numerals Coverter
## Kata
Write a function that takes an [Arabic number](https://en.wikipedia.org/wiki/Arabic_numerals) and returns its [Roman numeral](https://en.wikipedia.org/wiki/Roman_numerals) equivalent (as a string).

## Design
Use Test-Driven Development ([TDD](https://en.wikipedia.org/wiki/Test-driven_development)) and Transformation Priority Premise ([TPP](https://en.wikipedia.org/wiki/Transformation_Priority_Premise)) to drive the design of this code kata.

## Implementation Notes
Use [Modern C++](https://docs.microsoft.com/en-us/cpp/cpp/welcome-back-to-cpp-modern-cpp?view=vs-2019) and [Google Test](https://github.com/google/googletest) testing and mocking framework.
